# Architectural Decisions

This file is the durable, human-readable archive of architectural decisions. The Knowledge Graph is the live, queryable interface; this file is the versioned backup.

## Format

### ADR-NNN: Title
- **Date**: YYYY-MM-DD
- **Status**: Accepted | Superseded | Deprecated
- **Context**: What prompted the decision
- **Decision**: What was decided
- **Consequences**: What follows from the decision
- **KG Entity**: Name of the corresponding Knowledge Graph entity

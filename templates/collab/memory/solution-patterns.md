# Solution Patterns

Curated catalog of recurring solution patterns extracted from experience. The Knowledge Graph stores these as `solution_pattern` entities; this file is the human-readable archive.

## Format

### Pattern: Title
- **Applies when**: Conditions that indicate this pattern is relevant
- **Steps**:
  1. Step one
  2. Step two
- **Reference implementation**: File path and line numbers of an exemplar
- **KG Entity**: Name of the corresponding Knowledge Graph entity
